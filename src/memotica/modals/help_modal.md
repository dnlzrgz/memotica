### General Bindings

- F1: Show this screen.
- ctrl+q: Quit/Cancel.
- ctrl+b: Show/hide decks tree.
- escape: Close modal.
- ctrl+d: Toggle dark mode.
- ctrl+n: Open modal to add a new deck.
- ctrl+a: Open modal to add a new card.

### Decks

- backspace: Delete selected deck.
- ctrl-e: Edit selected deck.
